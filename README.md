# 🏏 IPL Mega Auction Analytics Dashboard

## 📊 Project Overview
A 3-page interactive Power BI dashboard that simulates the entire IPL player auction process.

**Tech Stack:** Power BI Desktop (DAX, Power Query), Excel

---

## 🔗 Dashboard Pages

### Page 1: Player Pool
- Analyze 30 players with batting, bowling & fielding stats
- Custom DAX measures: Performance Score & Form Indicator
- Player slicer with real-time stats update

![Page 1](page1.png)

### Page 2: Auction Tracker
- Select any team (CSK, MI, RCB, etc.)
- Place bids from ₹0.5 Cr to ₹30 Cr
- Real-time budget tracking (₹100 Cr per team)

![Page 2](page2.png)

### Page 3: Team Squad
- View final squads for all 10 teams
- Track total spent, remaining budget & slots left

![Page 3](page3.png)

---

## 🛠️ How to View

1. Download the `.pbix` file
2. Open with Power BI Desktop (free)
3. Navigate through 3 pages using bottom tabs

## 📈 Key Features

✅ 30 players with complete stats
✅ 10 teams with ₹100 Cr budget each
